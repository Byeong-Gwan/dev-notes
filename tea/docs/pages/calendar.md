# Calendar 페이지 문서

## 개요
달력 기반 일정/이벤트/주문 일정 관리 화면(추정).

## 주요 기능(추정)
- 달력 뷰 전환(월/주/일)
- 이벤트 목록/상세/추가

## 중요 로직/파일
- `tea/src/pages/calendar.html`
- 캘린더 관련 JS(해당 페이지에서 로드되는 스크립트)

## UX 포인트
- 반응형, 스크롤 고정 헤더, 오늘로 이동 버튼

## 개선 과제
- 키보드 내비게이션, 접근성 라벨링
- 드래그/드롭 편집, 시간대/로컬라이징

## 데모 시나리오
1) 월 뷰 → 2) 이벤트 클릭 → 3) 상세 확인/편집
