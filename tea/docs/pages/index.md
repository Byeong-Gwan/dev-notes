# Index(홈) 페이지 문서

## 개요
서비스 랜딩/개요 페이지로 주요 기능으로 진입하는 허브.

## 주요 기능
- 상단 메뉴/네비게이션
- 주요 페이지 링크 카드

## 중요 로직/파일
- `tea/src/pages/index.html`
- 공통 초기화: `tea/src/js/ui-init.js`, `tea/src/js/theme.js`

## UX 포인트
- 첫 인상: 핵심 가치/진입 동선 명확
- 반응형 그리드

## 개선 과제
- 히어로 섹션, 소개 카피, 데모 링크
- 라이트하우스 점수 개선(이미지 최적화)

## 데모 시나리오
1) 홈 → 2) 주요 메뉴 클릭
